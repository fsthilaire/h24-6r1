Communiquer avec l'utilisateur

Cette procédure explique comment un agent peut communiquer avec l'utilisateur à partir d'un billet.

Important

Tout le texte inscrit dans une communication est visible à l'utilisateur.

Procédure​

  1. Accéder au billet de l'utilisateur.

  2. Sélectionnez la section Répondre au message.

  3. Dans la zone de texte, inscrivez le message pour l'utilisateur.

    • Il s'agit d'un éditeur WYSIWYG. Il est possible d'incorporer des images.
    • Il est possible d'ajouter des fichiers.

    astuce

    Pour la réponse initiale, il est important de se présenter et poser les questions pour la collecte d'information.

  4. Appuyez sur le bouton Répondre au message pour envoyer la communication.
