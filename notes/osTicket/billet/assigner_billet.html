Assigner un billet

Cette procédure explique comment un agent peut accéder à un billet pour voir son contenu et pour le mettre à jour.

Procédure​

  1. Connectez-vous au panneau de l'agent.
  2. Cliquez sur l'onglet Ticket.

  1. Cochez le billet à assigner.

  2. Cliquez sur l'icône Assigner.

  3. Sélectionnez Agent dans la liste.

  4. Dans la liste Affecté à, sélectionner l'agent.

  5. Appuyez sur le bouton Assigner.

  6. Le billet aura le nom de l'agent dans la colonne Assigned To.
