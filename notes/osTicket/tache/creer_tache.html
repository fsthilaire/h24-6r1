Créer une tâche au billet

Cette procédure explique comment un agent peut créer une tâche pour un autre technicien dans un billet.

Procédure​

  1. Accéder au billet de l'utilisateur.

  2. Sélectionnez la section Tâches.

  3. Appuyez sur le bouton Ajouter une nouvelle tâche.

  4. Dans la fenêtre Ajouter une nouvelle tâche, il faut mettre l'information de la tâche.

    Description des champs

    • Title : Il faut décrire la tâche en quelques mots.

    • Description : Il faut décrire la demande ou le problème en détail.

      • Il s'agit d'un éditeur WYSIWYG. Il est possible d'incorporer des images.

      • Il est possible d'ajouter des fichiers.

    • Service : Le service qui s'occupe de la tâche. Toujours utiliser Support.

    • Affecté à : Permet d'assigner la tâche au membre responsable de l'équipe.

  5. Appuyez sur le bouton Créer une Tâche.
