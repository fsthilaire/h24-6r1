Ventoy

Ventoy est un logiciel qui permet de créer une unité de stockage démarrable avec plusieurs images.

La version présentée est la 1.0.95.

Pour plus d'information, veuillez visiter le site de Ventoy.

1. Téléchargement​

Téléchargez la version Windows de Ventoy 1.0.95.

Le programme se téléchargera automatiquement après quelques secondes.

2. Installation​

Il s'agit d'un logiciel portable.

Veuillez extraire le fichier zip dans un dossier.

Je recommande le dossier c:\outils\ventoy.

3. Créer une clé démarrable - Configuration de base​

  1. Insérez la clé USB dans l'ordinateur.

  2. Démarrez le programme Ventoy2Disk.exe en tant qu'administrateur.

  3. Sélectionnez la clé USB dans la liste déroulante.

  4. Dans le menu Option, décochez l'item Secure Boot Support.

  5. Dans le menu Option -> Partition Style, cochez GPT.

  6. Il est possible de faire des options spécifiques pour les partitions dans le menu Option -> Partition Configuration. Référez-vous à la section 6 du document pour connaitre les différentes configurations.

  7. Appuyez sur le bouton Install.

    Avertissement

    Si la clé contient déjà Ventoy, il y aura une fenêtre de confirmation additionnelle.

    Il faut inscrire YES en majuscule dans la zone de texte et il faut appuyer sur le bouton OK.

  8. Appuyez sur le bouton Oui pour confirmer l'avertissement que le contenu de la clé sera effacé.

  9. Appuyez sur le bouton Oui pour confirmer le 2e avertissement que le contenu de la clé sera effacé.

  10. Appuyez sur le bouton OK lorsque la création est terminée.

  1. La partie de droite indique la version de Ventoy de la clé sélectionnée.

4. Copie du ISO​

  1. Trouvez la lettre du lecteur de la partition Ventoy de la clé USB.

  1. Directement à la racine de la clé, il faut copier les fichiers ISO.

danger

Si vous voyez une partition nommée VTOYEFI en FAT de 32 Mo, il est important de ne pas l'utiliser.

5. Utilisation​

Vous devez démarrer l'ordinateur à partir de sur la clé USB.

  1. Sur les ordinateurs Dell, la touche est F12 permet de choisir l'unité de stockage pour le démarrage lors du démarrage de l'ordinateur.

    Avertissement

    Le nom de la clé USB sera différent en fonction du modèle utilisé.

    L'apparence du menu sera différente en fonction de l'ordinateur.

  2. Lorsque la clé est chargée en mémoire, il faut sélectionner le fichier ISO à utiliser dans la liste avec le clavier.

  3. Ensuite, il faut choisir l'item Boot in normal mode.

6. Configuration des partitions​

Vous devez appliquer ces options à l'étape 6 de la section 3.

Les options de configuration se retrouvent dans le menu Option -> Partition Configuration.

6.1. Système de fichiers de la partition Ventoy​

Il est possible de spécifier le système de fichiers de la partition Ventoy.

La partition Ventoy est celle qui contient les fichiers ISO.

  1. Dans la section File System, il est possible de sélectionner exFAT, NTFS, FAT32 ou UDF.

  2. Appuyez sur le bouton OK pour confirmer la configuration.

6.2. Partitions additionnelles​

Par défaut, la partition Ventoy prend tout l'espace disponible sur la clé.

Il est possible de réserver une partie de l'espace de la clé USB. Il y aura donc de l'espace non alloué à la fin de la clé.

  1. Cochez la case Preserve some space at the end of the disk.

  2. Sélectionnez l'unité de mesure dans la liste déroulante. Vous pouvez spécifier Gigaoctet (GB) ou Mégaoctet (MB).

  3. Dans la zone de texte, inscrivez la taille.

  4. Appuyez sur le bouton OK pour confirmer la configuration.

  5. Lorsque la clé est créée, il est possible de créer de nouvelles partitions sur l'espace non alloué de la clé.
