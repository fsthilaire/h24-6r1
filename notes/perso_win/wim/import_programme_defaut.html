Importer le profil des programmes par défaut

Prérequis​

  1. Avoir une clé USB démarrable avec le programme d'installation Windows ou Windows PE.

  2. Avoir un fichier de profil des programmes par défaut.

  3. L'ordinateur a été généralisé avec Sysprep.

Procédure​

  1. Démarrez l'ordinateur avec la clé démarrable d'installation Windows ou Windows PE.

    astuce

    Pour accéder à l'invite de commandes à partir du programme d'installation, appuyez sur les touches MAJ + F10 lorsque la fenêtre initiale est affichée.

  2. Identifiez la lettre du lecteur de l'unité de stockage qui recevra l'image et celle de la partition qui contient l'installation de Windows.

    Rappel
    diskpart
    list volume

  3. Exécutez la commande

    Dism /Image:[lettre partition windows] /Import-DefaultAppAssociations:[fichier]
    Exemple

    La lettre de la partition de Windows est C:.

    Le fichier du profil sera c:\default.xml.

    Dism /Image:c:\ /Import-DefaultAppAssociations:c:\default.xml

Référence​

https://learn.microsoft.com/fr-ca/windows-hardware/manufacture/desktop/export-or-import-default-application-associations?view=windows-11
