Demande Docusaurus

Dans ce laboratoire, vous devez réaliser une demande de Gaëtan Tanguay.

Le laboratoire peut se faire en équipe de 2.

Ce laboratoire vous permettra de vous familiariser avec l'outil Docusaurus. Cet outil devra être utilisé pour l'évaluation 1.

Demande initiale​

Voici la demande que Gaëtan vous écrit par courriel.

Bonjour l'informatique !

Nous avons perdu la source d'un projet Docusaurus pour l'équipe de maintenance. Il faut reproduire la source du projet pour avoir un site identique à celui qui est en production actuellement.

Il a uniquement quelques trucs sur le réseau de ce que le stagiaire avait fait.
 
Merci de reproduire le site et y apporter quelques modifications.

Bonne journée

Vous devez créer sa demande initiale dans osTicket.

Pensez à faire un rappel à Gaëtan d'utiliser le système de billet.

Marche à suivre​

À partir de la documentation officielle, vous devez reproduire le site Docusaurus de Gaëtan.

Le fichier note.txt contient des notes personnelles du stagiaire.

Les url doivent être identiques.

La structure du site doit être identique.

Le visuel pour chacune des pages doit être identique.

Finalisation de la demande​

Il faut fermer le billet adéquatement.

Vous devez écrire l'information importante que Gaëtan doit connaitre pour sa demande.

Vous devez inscrire l'information importante dans les notes internes pour le reste de l'équipe.
