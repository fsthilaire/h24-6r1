Faire une image personnalisée

Dans ce laboratoire, vous devez effectuer une image personnalisée pour l'installation de Windows.

Vous pouvez vous mettre en équipe de 2 pour faire ce laboratoire.

1. Matériel de l'étudiant​

Vous devez apporter votre disque interne et votre clé USB.

2. Matériel fourni​

Pour ce laboratoire, vous avez ce matériel.

  • Disque SSD qui contient :
    • ISO officiel de Windows 11 23H2 (multi édition)
    • Installation de Google Chrome Hors ligne
    • Installation de Windows Assessment and Deployment Kit (Windows ADK) Hors ligne.
  • Adaptateur USB à SATA
  • Boitier convertisseur de disque 2.5" à 3.5"

3. Clé USB Ventoy​

Avec votre clé USB personnelle, vous devez créer une clé USB démarrable avec Ventoy 1.0.95.

4. Nettoyer l'image d'installation​

Vous devez nettoyer l'image d'installation actuelle avant de procéder à l'installation.

Les ordinateurs du 1204 ont une licence OEM pour Windows Édition Famille.

Copiez le nouveau ISO sur votre clé USB.

5. Image de base​

Il faut installer Windows 11 Professionnel en mode Audit pour faire l'image personnalisée.

En premier lieu, il faut mettre à jour Windows et installer tous les pilotes.

Vous devez installer Google Chrome.

Il faut mettre Google Chrome comme navigateur par défaut.

Les fichiers PDF doivent utiliser Google Chrome comme logiciel par défaut.

Vous devez créer le compte administrateur local ci-dessous :

  • Nom : d'utilisateur : locadmin
  • Mot de passe : Cegep-6R1!2023
  • Activez l'option : Le mot de passe n'expire jamais.

L'image doit s'appeler Windows 11 Pro - 23H2 - 1204 - Base.

Le fichier ISO doit s'appeler Win11_Pro_23H3_1204_Base.iso.

6. Tester l'installation de Windows​

Lorsque votre image est prête, vous devez installer Windows avec votre nouvelle image pour tester l'installation.

Connectez-vous avec le compte locadmin.

Renommez l'ordinateur en fonction du standard de l'entreprise.

Rejoignez le domaine de l'entreprise.

Ajoutez le groupe Techniciens N1 du domaine dans le groupe local Administrateurs.

Connectez-vous avec votre compte de l'entreprise.

Installez les logiciels ci-dessous :

  • RSAT : outils Services de domaine Active Directory et Lightweight Directory Services

    astuce

    Il faut installer RSAT par les fonctionnalités facultatives de Windows.

  • Windows Assessment and Deployment Kit (Windows ADK)
